quiz.js 파일에서 QUESTIONS 배열을 읽고, 각 문제의 question 필드에 모호한 표현('가장', '최초', '최대', '최다', '최소', '최장', '최고', '최저', '처음')이 포함된 문제를 찾아라.

$ARGUMENTS 값이 있으면 해당 카테고리(category 필드)의 문제만 검증하고, 없으면 전체 문제를 검증하라.
카테고리 코드와 이름 대응: korean-history(한국사), science(과학), geography(지리), general(일반상식)
$ARGUMENTS에 한글 이름이 입력되면 해당 코드로 매핑하라.

결과는 아래 형식으로 출력하라:

## 모호한 표현이 포함된 문제 목록
(카테고리가 지정된 경우 "── [카테고리명] ──" 헤더를 표시하라)

| ID | 카테고리 | 질문 | 포함된 표현 | 기준 명시 여부 |
|----|---------|------|------------|--------------|
| ... | ... | ... | ... | ✅ 명시됨 / ⚠️ 미명시 |

기준 명시 여부 판단 기준:
- ✅ 명시됨: 질문 안에 "해발 고도 기준", "면적 기준", "전통적인 측정 기준" 등 측정 기준이나 시점이 명확히 적혀 있는 경우
- ⚠️ 미명시: 모호한 표현만 있고 어떤 기준으로 판단하는지 명시되지 않은 경우

목록 출력 후, ⚠️ 미명시 항목 각각에 대해 어떤 기준을 명시해야 하는지와 권장 수정안을 제안하라.
